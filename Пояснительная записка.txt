Мангаридер. Приложение для онлайн и оффлайн чтения манги.

Функции:
- Поиск по названию из каталога Desu.
- Сортировка по названию, популярности, типу, жанрам.
- Скачивание, для чтения в автономном режиме.

Используемые библиотеки: pyqt5, requests, keyboard, sqlite3, pathlib, sys, os, threading.

При запкске программы появляется главное окно.
На главном окне отображеется манга. На окне есть кнопки для переключения страниц, поле для текста в которое можно
ввести название манги для поиска. Есть фильтры по названию, популярности, типу, жанрам.(манга берется из каталога Desu("https://desu.me")
с помощью библиотеки "requests"). Информация о манге добавляется базу данных(для этого используется функция "manga_add"
в файле static.py).
Двойное нажатие на мангу открывает окно с информацией о ней. На окне отображается preview, название, описание,
рейтинг и список глав. На окне есть кнопка при нажатии которой манга добавляется в избранное, при повторном нажатии манга удаляется из избранного.
(информация о главах добавляется в базу данных "chapters_add").
На главном окне есть кнопки переключения между каталогом манги и списком избранного. В списке избранного отображатся манга добавленная
из списка глав. Ее можно скачать полностью при помощи кнопки загрузки. Манга скачивается в фоне при помощи медода "download_all" в классе "Desu".
Мангу из избранного можно открыть в оффлайн режиме если она скачана.
При двойном нажатии на главу открывается читалка на полный экран. При открытии в базу данных добавляется информация о страницах в открытой главе
"images_add" и запускается процесс скичивания всех страниц в главе при помощи библиотеки "threading"
В центре отображаются страницы, в нижней части есть книпки для переключения страниц и глав, также их можно переключать с помощью стрелок
на клавиатуре(считывание нажатий с клавиатуры осуществляется с помощью библиотеки "keyboard" в классе "Reader"). Переключение
осущетвляется при помощи методов "change_page_reader" и "change_chapter_reader".
